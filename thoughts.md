# Thoughts

我的想法、eureka 瞬间、半成品的 take。不是外部信息——那些去 `inbox.md`。

**写法：** 每条简短，三个字段即可。

**触发方式：** 在 `inbox.md` 中用 `💡` 或 `[idea]` 前缀开头；digest 时会自动路由到这里，不会写入 topics/。

**检索：** 每次你在对话里提到项目（qf / remi / ckg / clinamen 等），我会扫描此文件中 `可能关联` 字段匹配的条目，在聊天里列给你。
